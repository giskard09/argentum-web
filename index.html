what is a trail?

A TrailRecord is a single post-execution anchor: one agent action, one outcome, one immutable record on-chain. Anchoring happens after execution — Mycelium reads the outcome, not the action itself.

can I verify a trail without Mycelium infrastructure?

Yes. Any auditor with the evidence file and the tx hash can verify independently. No access to our runtime required.

how do I start?

Free tier starts immediately — no signup. See argentum-core for integration docs. Enterprise: @rama_contact_bot on Telegram.
